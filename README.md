# Lumos University – Static Website Project

This repository contains a simple static website for a **fictional** university called **Lumos University**.  

The goal of this project was to demonstrate a foundational understanding of HTML and CSS in the context of layout, color, and responsiveness.

---

## Project Overview

**Lumos University** is a fictional university.

This site is split into the following parts:

1. Home page 
2. About
3. Academics
4. Admissions
5. Athletics
6. Clubs
7. Contact Us
8. Events
9. Privacy Policy
10. Student Life
11. Tuition

---

## Tech Stack

- **HTML5** – Page structure and content  
- **CSS3** – Layout, styling, and responsiveness  

---

## How to Run the Project

1. Clone or download the repository  
2. Open `index.html` in a browser  
3. Navigate between pages using links or by opening the files directly
